## Tecnologias y dependencias
- React js
- Redux Toolkit
- Axios
- Material UI
- React Router
- Yup
- Formik
- Chart js
- Sweet Alert

## Instalación



Primero instalamos las dependencias y luego iniciamos el servidor local.

```sh
$ cd alkybank
```

```sh
$ npm i
```

```sh
$ npm start
```
